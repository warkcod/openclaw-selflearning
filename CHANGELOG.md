# Changelog

All notable changes to this project will be documented in this file.

The format is loosely based on Keep a Changelog, with versioned release notes added as the project starts publishing tagged releases.

## [0.1.0] - 2026-04-16

### Added

- OpenClaw-native self-learning plugin entry and configuration schema
- File-backed per-agent learning store
- Selective post-turn review trigger
- Strict JSON review worker parsing and validation
- Memory, transcript recall, and skill asset model
- Patch proposal governance and skill revision workflows
- Evolution traces, stale transitions, suppression, and re-promotion
- Learning bundle export and import
- `/selflearn` command surface
- Host-loader smoke and host-runtime integration tests
