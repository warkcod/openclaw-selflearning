# Code of Conduct

## Our Pledge

We want this project to be a respectful, technically serious, and welcoming place for contributors and users.

Participants are expected to:

- discuss ideas directly and respectfully
- focus criticism on technical work, not people
- assume good intent while remaining rigorous
- avoid harassment, abuse, or exclusionary behavior

## Expected Behavior

- Be respectful in public and private communication.
- Give technical feedback with evidence and clear reasoning.
- Accept correction without escalation.
- Keep discussions relevant to the project.

## Unacceptable Behavior

- Harassment, personal attacks, or intimidation
- Discriminatory language or conduct
- Publishing private information without permission
- Repeatedly derailing technical discussions

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation when behavior violates this code of conduct.

## Scope

This code of conduct applies in:

- GitHub issues
- pull requests
- discussions
- project chat or coordination channels used for this repository

## Reporting

If you encounter unacceptable behavior, open a private security or maintainer contact route rather than escalating publicly where possible.
